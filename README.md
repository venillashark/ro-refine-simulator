# ro-refine-simulator
RO 精煉戰情室 - 升階精煉模擬與歐非判定工具 (Refine Simulator with Luck Analysis) 專為 Ragnarok Online 開發的精煉模擬器，支援二階裝備、即時物價計算及基於 100 次模擬基準的「歐非判定」系統。

# 🛠️ RO 精煉戰情室 (RO Refine Simulator)

![Version](https://img.shields.io/badge/version-6.5.1-blue)
![Language](https://img.shields.io/badge/language-HTML%2FJS-orange)
![Game](https://img.shields.io/badge/Game-Ragnarok%20Online-red)

這是一款專為《RO 仙境傳說 Online》設計的高級精煉模擬工具，特別針對 **二級防具（Grade 系統）** 的精煉流程與升階機率進行優化，並內建科學化的「歐非判定」系統。

## ✨ 核心特色

- **二級防具精煉全支援**：支援從無階級到 A 階的完整升階流程，自動切換消耗材料（乙太天藍、黃、紫、琥珀寶石）。
- **智能物價同步**：內建原料成本計算器，只需輸入星塵與魔石價格，一鍵同步所有加工材料的製作成本。
- **鐵匠祝福門檻**：可自定義從 +7 或 +10 開始消耗鐵匠祝福，精確計算 Zeny 與 TWD 支出。
- **🧪 歐非判定系統**：獨創「模擬基準」功能。系統會預跑 100 次完整精煉流程取平均值，對比你當前的實際消耗，即時判定你是「天選之子」還是「非洲大酋長」。
- **結案檔案庫**：支援多筆精煉紀錄存檔，方便對比不同裝備的精煉開銷。

## 🚀 快速上手

1. **設定物價**：在左側填入當前的 R 幣匯率與各類材料市價（或使用同步功能）。
2. **執行模擬基準**：在點擊精煉前，先設定目標（例如：目標 A 階 +12），按背景執行 100 次模擬。
3. **開始點裝**：點擊「精煉成功/失敗」或「升階成功/失敗」，系統將即時計算累計成本與運氣狀態。
4. **存檔結案**：完成後點擊「結案存檔」，紀錄將自動保存至下方的檔案庫中。

## 📊 精煉邏輯說明

- **精煉機率**：嚴格遵循官方公佈之二階防具精煉機率表。
- **失敗判定**：
  - 無鐵匠祝福時：失敗將面臨精煉值下降 1~3 階之懲罰。
  - 設有鐵匠祝福時：保護精煉值不下降，但會消耗對應數量的祝福。
- **升階規則**：自動判定當前等級是否達到升階門檻（D 階 +9, C 階 +10, B/A 階 +11）。
